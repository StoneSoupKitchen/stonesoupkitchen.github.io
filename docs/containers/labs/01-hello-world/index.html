Containers

Lab #1: Hello, World!#

Overview#

Get started using Docker with everyone's favorite exercise: Hello, World!

Challenge#

Create a Docker image from debian:stable-slim named helloworld that prints the string "Hello, World!" when it is run.

Solution#

Click to revealCreate a `Dockerfile` with these contents:
FROM debian:stable-slim
CMD [ "echo", "Hello, World!" ]

Build the container:

docker build -t helloworld .

After Docker pulls down the container from Docker Hub and builds the container, run it to verify that we've completed this challenge:

docker run helloworld
